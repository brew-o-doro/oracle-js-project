<h1>Oracle<h1>
<h4> Oracle is an anonymous posting site similar to whisper without the pictures  </h4>
<h2> Features </h2> 
<ul>
  <li>Post a message!</li>
  <li>Reply to a post with a comment!</li>
</ul>
<h2> Tech Stack </h2>
<h3> Backend </h3
<ul>
  <li>Ruby [2.6.1]</li>
  <li>Rack-Cors</li>
</ul>
<h3> Frontend </h3>
<ul>
  <li>Vanilla JavaScript [ES6]</li>
  <li>Custom CSS3 styling</li>
</ul>
